# AI Chatbot Website 🤖

A simple AI chatbot website built using Flask, HTML, CSS, and JavaScript.

## Features
- Chat interface
- AI responses
- Responsive UI
- Flask backend

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript

## Run Locally

Install dependencies:
pip install -r requirements.txt

Run:
python app.py

Open:
http://127.0.0.1:5000
